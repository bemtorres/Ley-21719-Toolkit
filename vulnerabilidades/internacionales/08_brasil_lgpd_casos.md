# Caso Real 8: Brasil — Primeras Multas y Brechas bajo la LGPD (2023-2025)


## Resumen

Brasil, con su **Lei Geral de Proteção de Dados (LGPD)** —inspirada en el GDPR—, comenzó a aplicar multas efectivas en **2023** a través de la Autoridade Nacional de Proteção de Dados (ANPD). A diferencia de Chile (que hasta 2026 carecía de multas reales), Brasil ya ha sancionado a empresas públicas y privadas, incluyendo la primera multa LGPD a **Telekall Infoservice** en 2023 y una filtración masiva en **IAMSPE** que expuso datos de **1,5 millones de funcionarios públicos**.

## Caso A: Telekall Infoservice — Primera Multa LGPD (Julio 2023)

| Aspecto | Detalle |
| :--- | :--- |
| **Empresa** | Telekall Infoservice (telemarketing) |
| **Violación** | Tratamiento de datos sin base legal, falta de DPO, obstrucción a la investigación |
| **Datos** | Números de teléfono y WhatsApp de ciudadanos de Ubatuba (SP) usados para fines electorales |
| **Multa** | BRL 14.400 (~$2.960 USD) — 2% del faturamento, tope para microempresas |
| **Lección** | La ANPD confirmó que la LGPD aplica a empresas de **todos los tamaños**, no solo a grandes corporaciones |

## Caso B: IAMSPE — Brecha en el Sector Público (2023)

| Aspecto | Detalle |
| :--- | :--- |
| **Institución** | IAMSPE (Instituto de Asistencia Médica al Servidor Público del Estado de São Paulo) |
| **Brecha** | Ciberataque expuso datos de salud de **1,5 millones de funcionarios públicos** |
| **Datos** | Datos médicos, diagnósticos, tratamientos, información de identificación personal |
| **Causa** | Vulnerabilidades en API; controles de seguridad inadecuados |
| **Notificación** | IAMSPE tardó **3 meses** en notificar la brecha a la ANPD |
| **Sanción** | Advertencia pública y orden de implementar ISO 27001 (sin multa por ser entidad pública) |

## Caso C: Meta — Prohibición de Uso de Datos para IA (2024)

| Aspecto | Detalle |
| :--- | :--- |
| **Base legal** | ANPD prohibió a Meta usar datos de Facebook/Instagram para entrenar modelos de IA |
| **Violación** | Falta de consentimiento válido para el tratamiento de datos con fines de IA generativa |
| **Sanción** | Prohibición operacional + multa diaria de BRL 50.000 (~$10.000 USD) por incumplimiento |

## Comparativa con Chile

| Aspecto | Brasil (LGPD) | Chile (Ley 21.719) |
| :--- | :--- | :--- |
| **Vigencia** | 2020 (multas desde 2023) | Diciembre 2026 |
| **Primera multa** | BRL 14.400 (Telekall, 2023) | Aún no aplicada |
| **Máxima multa** | 2% fact. Brasil (R$ 50M tope) | 20.000 UTM (~$1.400 MM) o 4% ingresos |
| **Entidad pública** | Multas prohibidas | Sanciones administrativas posibles |
| **Casos emblemáticos** | Telekall, IAMSPE, Meta | Caja Los Andes (sin multa, pre-ley) |

## Lecciones para la Ley 21.719

1. **La ANPD comenzó con multas pequeñas pero simbólicas:** La primera multa LGPD fue solo de BRL 14.400. Esto demuestra que la APDP chilena probablemente empezará con casos de alto perfil pero montos moderados, aumentando progresivamente.
2. **Sector público no está exento:** Aunque IAMSPE no recibió multa, la fiscalización y exposición pública fueron igualmente dañinas.
3. **Notificación oportuna de brechas:** Los 3 meses de retraso de IAMSPE agravaron su sanción. Chile exige 72 horas.
4. **IA y datos personales:** El caso Meta/ANPD muestra que los reguladores latinoamericanos están fiscalizando el uso de datos para entrenar modelos de IA, incluso antes de que la ley esté completamente implementada.
