# TEMPLATE: FORMULARIO DE CONSENTIMIENTO GRANULAR
## Ley 21.719 - Consentimiento libre, específico, informado e inequívoco

### Instrucciones de uso:
- Cada finalidad debe tener su propia casilla de verificación (checkbox).
- Ninguna casilla debe venir premarcada por defecto.
- El titular debe poder aceptar o rechazar cada finalidad de forma independiente.
- Incluir enlace a la Política de Privacidad actualizada.

---

## Sección 1: Consentimiento para el Tratamiento de Datos Personales

```
[NOMBRE DE LA ORGANIZACIÓN]

Para poder prestar nuestros servicios, necesitamos tratar sus datos personales.
Por favor, seleccione las opciones que autoriza:

[ ] 1. DATOS BÁSICOS DE SERVICIO (Requerido para el servicio)
    Autorizo el tratamiento de mis datos de identificación (nombre, RUN, correo,
    teléfono) con la finalidad de gestionar mi cuenta, procesar mis solicitudes
    y prestar el servicio contratado.
    Base legal: Ejecución de un Contrato.

[ ] 2. DATOS SENSIBLES (Requerido para el servicio que incluye datos de salud)
    Autorizo el tratamiento de mis datos de salud declarados, exclusivamente
    para los fines del servicio solicitado.
    Base legal: Consentimiento Explícito.

[ ] 3. COMUNICACIONES COMERCIALES (Opcional)
    Autorizo el envío de comunicaciones promocionales, ofertas y boletines
    informativos relacionados con productos o servicios similares a los
    contratados.
    Base legal: Consentimiento.
    Puede revocar este consentimiento en cualquier momento.

[ ] 4. CESIÓN DE DATOS A TERCEROS (Opcional)
    Autorizo la comunicación de mis datos a empresas del grupo económico
    para fines comerciales.
    Base legal: Consentimiento.

[ ] 5. TRANSFERENCIA INTERNACIONAL (Opcional / según aplique)
    Autorizo la transferencia de mis datos a servidores ubicados en el
    extranjero, bajo las salvaguardas establecidas en la Política de
    Privacidad.
    Base legal: Consentimiento.
    Países de destino: [EE.UU., Argentina, etc.]
    Salvaguardas: Cláusulas Contractuales Tipo (CCT)

[ ] 6. DATOS DE MENORES DE 14 AÑOS (Solo si aplica)
    En mi calidad de apoderado/a del menor [NOMBRE], autorizo el tratamiento
    de sus datos personales en los términos descritos en esta solicitud.
    Acompaño copia de mi cédula de identidad para acreditar la filiación.
```
