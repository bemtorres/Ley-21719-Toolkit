## Estructura de cada caso

Cada caso incluye:

- **Resumen** — Qué pasó, cuándo y quiénes fueron afectados
- **Datos expuestos** — Tipos y volúmenes de datos comprometidos
- **Causa técnica/legal** — Cómo ocurrió la brecha
- **Multa / Sanción** — Detalle de la penalidad aplicada
- **Lecciones para la Ley 21.719** — Análisis de impacto en el contexto chileno
- **Referencias** — Fuentes oficiales y reportes de prensa

## Propósito

Estos casos no son hipotéticos: representan riesgos reales que enfrentan las organizaciones chilenas. El análisis de cada caso permite extraer lecciones concretas para la adecuación a la Ley 21.719 antes del **1 de diciembre de 2026**.
